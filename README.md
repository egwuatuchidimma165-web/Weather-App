#  Responsive Weather-App
### Project Description 
I  built this responsive weather application using  vanilla HTML5, CSS3, and modern asynchronous javaScript(ES6) to fetch real-time weather data from Open-Meteo API. T he project features a clean user interface that dynamically converts raw numerical data such as WMO weather codes and UV indicsee into user-friendly text categories(high, moderate).
